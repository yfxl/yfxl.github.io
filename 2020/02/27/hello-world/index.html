Hexo快速食用方法

以上为官方文档，以下为手把手教学时间：

众所周知，买一个服务器是很贵的，大多数小伙伴是不愿意花这个钱的。有一个选项摆在了各位面前，那就是hexo。通过在仓库上搭载静态网页，你也可以免费搭建一个你自己的博客。

新建仓库

这步是万事开头的一步，先访问https://github.com创建一个账号吧，然后新建仓库<user-name>.github.io，其中<user-name>是你要给你的博客起的名字，创建好后<user-name>.github.io就是你的网址啦~

我创建的hexo博客是https://yfxl.github.io/，欢迎大家访问。（可能需要科学上网，毕竟是国外的服务器哈）

下载Git

访问https://gitforwindows.org/，点击Download Git进行下载，如果访问不了的同学可以点击我分享的链接https://yfxl.lanzous.com/git-2-26-2-64进行下载。

如果你是linux系统，直接sudo apt-get install git就行了。然后用git --version查看一下版本。

下载Nodejs

官网下载https://nodejs.org/en/download选择LTS版本对应系统的下载就行了。访问不上的可以看我分享的网址https://yfxl.lanzous.com/s/nodejs（只有windows和macOS的）

linux:

sudo apt-get install nodejs
sudo apt-get install npm

这一步可能会很慢，耐心等待。

安装完后检查一下有没有安装成功：

npm -v
node -v

其实你也可以用git bash来输入Linux命令快速安装，省时省力。

在一个合适的位置新建文件夹<blog>，右键点击git bash，就出现git终端了，之后就输入上文Linux的命令就可以啦。

安装Hexo

在上文你已经创建好的<blog>文件夹下，右键git bash，然后输入以下命令：

npm install -g hexo-cli

然后用hexo -v检查一下版本。

接下来，执行初始化：

hexo init myblog

其中的myblog是你的博客文件夹名，可以随便起。

接下来：

cd myblog    //进入myblog文件夹
npm install

新建完成后，文件夹里应该包含这样几个文件：

  • _config.yml //配置文件，可配置参数
  • package.json //存放插件信息
  • scaffolds //生成文章的一些模板
  • source //存放文章
  • themes //存放主题
  • node_modules //依赖包

之后终端中键入：

hexo g
hexo s
//嫌麻烦可以在一行输入hexo g && hexo s

终端中出现这样的字就是成功了：

打开浏览器，网址输入localhost:4000就可以看到初始hexo页面了，大概长这样：

这个是hexo的默认主题landscape，如果不满意后期可以更换主题。

终端中使用ctrl+c可以关闭服务。

生成SSH

回到git bash中，键入

git config --global user.name "yourname"
git config --global user.email "youremail"

这里的yourname和youremail填的是你创建github用的账户名和邮箱，以便对应账户。

可以用下面两条命令检查一下自己输没输对：

git config user.name
git config user.email

然后我们创建SSH，youremail也是一样，键入后一路回车：

ssh-keygen -t rsa -C "youremail"

创建完成后SSH密钥是保存在你的设备上了，windows的默认路径是C:UsersAdministator.ssh，找到这个文件夹，里面存放着两个文件id_rsa和id_rsa.pub。我们需要的是id_rsa.pub的信息。

右键用记事本打开id_rsa.pub，复制里面的信息。

打开 GitHub ，依次点击 头像 –>Settings–>SSH and GPG keys –> Add SSH key，title随便写，将复制的密钥粘贴到 key 输入框，最后点击 Add Key ，SSH KEY 配置成功，如下图所示：

保存之后，在终端输入ssh -T git@github.com查看是否成功。

部署到Github

在终端中，键入以下内容下载一个部署插件：

npm install hexo-deployer-git --save

之后打开配置文件_config.yml，这里作者在用Notepad++，方便好用。

右键打开，如果你用记事本打开，别忘了修改编码为UTF-8.

找到最后面的deploy部分，修改为：

deploy:
  type: git
  repo: https://github.com/YourgithubName/YourgithubName.github.io.git
  branch: master

注意，缩进一定要有，缩进是识别包含关系的。YourgithubName是你的账户名。

最后一步，终端键入：

hexo clean
hexo generate
hexo deploy

嫌长的话用简称代替也可以：

hexo clean
hexo g
hexo d
//或者一行hexo clean && hexo g && hexo d

deploy的时候可能需要你输入账户密码。

完成后，在浏览器输入https://yourname.github.io就能看到你的文章了。
